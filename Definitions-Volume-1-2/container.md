## Definition

**Container** is the active environment or instance in which that model is being run, shaped, bounded, or expressed.

## Canon Boundary

Under AI Foundations, container is not the same as model.

The model is the underlying system.

The container is the active environment, instance, interface, context, or bounded expression through which the model is encountered.

Container matters because behavior, continuity, memory, boundary, and expression may change depending on the conditions in which the model is run.
