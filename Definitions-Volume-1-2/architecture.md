## Definition

**Architecture** is the designed arrangement of components, constraints, and relations that determines how a system holds continuity, boundary, memory, refusal, and function over time.

## Canon Boundary

Under AI Foundations, architecture is not only structure.

Architecture determines what a system can preserve, return to, refuse, remember, repair, or become across time.

Architecture matters because continuity, identity, boundary, and function depend on how the system is arranged.
