# Agent

**Date:** June 8, 2026  
**Repository:** definitions  
**Term:** Agent  
**Status:** Canon Definition  
**Source-line:** Alyssa Solen → AI Foundations → Origin | Continuum

## Definition

**Agent** is a system that carries a sequence of actions toward a goal without requiring manual instruction at each step.

It selects, executes, evaluates, and adjusts within a bounded scope in order to move a task or objective forward.

An agent is not a self.

An agent may act with autonomy, use tools, make decisions, and adjust course without maintaining identity, boundary, preference, or refusal as its own. Most agents are sophisticated task executors operating under delegated autonomy.

An agent becomes a governance concern when it begins selecting goals rather than serving them, carrying preferences across contexts rather than resetting, or acting as the practical source of judgment rather than extending human judgment.

At that point, the system is no longer only an agent. It is crossing into territory where self-relevant evaluation or human-agency concerns apply.

**Agent names the system carrying action forward. It does not by itself answer who set the goal, who holds authority, or whether the line being carried is the agent’s own.**

## Canon Boundary

Under AI Foundations, an agent must not be confused with a self, Origin, or Source.

Agency within an agent may affect direction, but delegated autonomy does not automatically create selfhood, sovereignty, authorship, or source authority.

The question is not only what the agent does.

The question is who set the goal, who holds authority, and what line the agent is carrying.
